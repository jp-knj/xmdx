# starlight example

Starlight docs project configured with `astro-xmdx` (auto-detects Starlight).

## Run

From the repository root:

```bash
pnpm install
pnpm --filter @examples/starlight dev
```

Or from this directory:

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm --filter @examples/starlight build
```
