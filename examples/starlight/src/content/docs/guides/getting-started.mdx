## Installation

<Steps>

1. Install the required packages:

   ```bash
   pnpm add astro-xmdx
   ```

2. Add xmdx to your Astro config:

   ```js
   import xmdx from 'astro-xmdx';
   import { starlightPreset } from 'astro-xmdx/presets';

   export default defineConfig({
     integrations: [
       starlight({ title: 'My Docs' }),
       xmdx({ presets: [starlightPreset()] }),
     ],
   });
   ```

3. Run the dev server to confirm everything works:

   ```bash
   pnpm dev
   ```

</Steps>

:::caution
Make sure to add `xmdx()` **after** `starlight()` in the integrations array.
:::
