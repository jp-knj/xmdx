Welcome to the xmdx Starlight example. This site demonstrates the `starlightPreset()` integration.

## Directives

:::note
xmdx replaces the default MDX compiler with a high-performance Rust-based pipeline.
:::

:::tip[Performance]
The Rust-based compiler is significantly faster than the default JavaScript-based MDX pipeline.
:::

:::caution[Integration Order]
Make sure to add `xmdx()` **after** `starlight()` in the integrations array.
:::

:::danger
Do not remove the `starlightPreset()` from your config or Starlight components will stop rendering.
:::

## Code Block

```js title="astro.config.mjs"
import xmdx from 'astro-xmdx';
import { starlightPreset } from 'astro-xmdx/presets';

export default defineConfig({
  integrations: [
    starlight({ title: 'My Docs' }),
    xmdx({ presets: [starlightPreset()] }),
  ],
});
```
