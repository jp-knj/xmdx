# Welcome to xmdx

This is a minimal Astro site powered by **xmdx**.

## Features

xmdx compiles MDX using a high-performance Rust-based pipeline.

```js
import xmdx from 'astro-xmdx';

export default defineConfig({
  integrations: [xmdx()],
});
```

## Getting Started

Install the package and add it to your Astro config.
