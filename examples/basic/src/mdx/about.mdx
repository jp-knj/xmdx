# About This Example

This is a basic example demonstrating **astro-xmdx** integration with Astro.

The project shows how to use xmdx as a drop-in replacement for the default MDX compiler.

## Key Benefits

- High-performance Rust-based compilation
- Compatible with existing Astro MDX content
- Supports frontmatter, headings, and code blocks

## Links

- [Astro Documentation](https://docs.astro.build)
- [xmdx Repository](https://github.com/jp-knj/xmdx)

## Notes

1. xmdx processes `.mdx` files automatically
2. Frontmatter is parsed with `gray-matter`
3. Headings are extracted for table of contents
